# Max Feedback Bot (MVP)

Простой бот обратной связи для платформы Max.ru. 
Позволяет получать сообщения от пользователей и отвечать им через личный чат с ботом.

Аналог Olgram в минималистичном исполнении для Max.

## Возможности

- 📩 **Пересылка сообщений:** Все сообщения от пользователей пересылаются владельцу бота.
- 💬 **Ответы через Reply:** Владелец может ответить конкретному пользователю, используя функцию "Ответить" (Reply) на пересланном сообщении.
- 📊 **Статистика:** Команда `/stats` для просмотра количества пользователей.
- 💾 **База данных:** SQLite для сохранения истории и связок сообщений.

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/ваш-юзернейм/max-feedback-bot.git
cd max-feedback-bot
```

### 2. Установка зависимостей

```bash
npm install
```

### 3. Настройка окружения

Создайте файл `.env` в корне проекта и добавьте туда следующие переменные:

```env
BOT_TOKEN=ваш_токен_от_бота_Max
OWNER_ID=ваш_user_id
```

- `BOT_TOKEN`: можно получить при регистрации бота на [dev.max.ru](https://dev.max.ru).
- `OWNER_ID`: ваш цифровой ID в Max (бот напишет его вам при команде `/start` в первом варианте кода, или можно узнать через @username_to_id_bot, если такой есть, или посмотреть в логах).

### 4. Запуск

```bash
npm start
```

## Использование

1. **Пользователь** пишет боту `/start` или любое сообщение.
2. **Вы (Владелец)** получаете сообщение вида:
   > 📩 **Сообщение от Иван** (ID: 123456):
   >
   > Привет!
3. Чтобы ответить, нажмите на это сообщение функцию **"Ответить" (Reply)** и напишите текст.
4. Бот отправит ваш ответ пользователю.

Если вы просто напишете сообщение в чат (без Reply), оно уйдет **последнему** активному пользователю.

## Команды

- `/start` - Начало работы.
- `/stats` - Статистика по пользователям (доступна только владельцу).

## Технологии

- [Node.js](https://nodejs.org/)
- [@maxhub/max-bot-api](https://www.npmjs.com/package/@maxhub/max-bot-api) - Официальная библиотека Max API.
- [SQLite](https://www.sqlite.org/) - Локальная база данных.
